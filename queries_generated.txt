1. what are is the average price of all the products?

pipeline = [
{
"$group": {
 "_id": None,
"averagePrice": {
 "$avg": "$Price"
 }
}
}
]
db.Products.aggregate(pipeline)

4. Find all products with a rating below 4.5 that have more than 200 reviews and are offered by the brand 'Nike' or 
'Sony'.

Query generated:

pipeline = [
{
"$match": {
"Rating": {
"$lt": 4.5
},
"ReviewCount": {
"$gt": 200
},
"Brand": {
"$in": ["Nike", "Sony"]
}
}
}
]

db.Products.aggregate(pipeline)
